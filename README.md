# Reservations

An enhanced reservations workspace for TREK trip pages.

## Screenshots

![screenshot](./docs/screenshot.png)

## What it does

Keep transportation and bookings together in one trip-page workspace.

- transportation: flights, trains, buses, cars, taxis, bicycles, cruises,
  ferries, transit, and other transport
- bookings: hotels, restaurants, events, tours, and other reservations

## Setup

Install dependencies and start the client builder plus SDK dev server:

```bash
npm install
npm run dev
```

For a release, run `npm run validate` and `npm run pack`.
